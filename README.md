# Controls for handling application
_Camera Controls
- W: Move forwards
- S: Move backwards
- A: Strafe left
- D: Strafe right
- R: Reset camera to default position

_Pinball Controls
- ESC: Exit application
- Space: Hold to use plunger, and release to release ball
- Left Arrow: Left Flipper
- Right Arrow: Right Flipper
- P: Toggle Debug mode
